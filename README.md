# Contact Book

## 📌 About the Project

The Contact Book is a simple Python application that allows users to store and manage contact information through a menu-driven interface.

## ✨ Features

* Add a new contact
* View all contacts
* Search for a contact by name
* Delete a contact
* Exit the application

## 🛠️ Technologies Used

* Python

## 🎯 Learning Outcomes

This project helped me learn:

* Python dictionaries
* Loops and conditional statements
* Menu-driven programming
* Basic data management
